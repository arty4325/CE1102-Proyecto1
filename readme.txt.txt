Se utilizo Python 3.9.9

Se tiene que instalar la biblioteca keyboard y pygame para poder ejecutar el juego 

Ademas los controles del juego son los siguientes
Con la tecla w se puede acelerar, con la tecla s se puede frenar 
Con la tecla espacio se puede cambiar de marhca
Con la tecla a el vehiculo se mueve a la Izquierda
Con la tecla d el vehiculo se mueve a la Derecha
